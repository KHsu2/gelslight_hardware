* Colored Filament: `finger_gelsight_updated/shell_back.sldprt`
* Clear PLA: `finger_gelsight_updated/gelsight_corefilled_105.sldprt`

## Laser Cut Files

* Acrylic: `finger_gelsight_production/gelsight_corefilled_acrylic.ai`
